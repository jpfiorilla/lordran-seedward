# Reference materials

Use these when editing areas, connections, nodes, or fog gates so the data matches the game.

## Fog doors / fog gates

**Fandom wiki list of fog doors (by area):**  
https://darksouls.fandom.com/wiki/Fog_Door

Use when defining or updating `FOG_GATES` in `fogGates.ts`. The wiki lists “Other fog doors”, “Lordran fog doors”, and “Oolacile fog doors” by location.
